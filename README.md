# WB → УПД

Утилита для российских ИП и ООО, продающих товар на **Wildberries** по схеме
выкупа (когда WB выкупает товар у продавца под свою реализацию в странах СНГ и
присылает **Уведомление о выкупе**).

Раз в неделю (или по нажатию кнопки):

1. Забирает из WB API уведомления о выкупе (категория `redeem-notification`)
2. Парсит xlsx из архива, считает суммы и НДС
3. Генерирует **УПД** в формате ФНС 5.03, функция СЧФДОП (кодировка
   windows-1251) — файл, готовый к загрузке в Диадок / СБИС / Такском
4. Складывает все XML в одну папку и умеет отдать их одним zip-архивом

> ⚠️ **УПД загружаются в Диадок вручную** (перетаскиванием в веб-кабинет).
> API-лицензия Диадока стоит ~27 000 ₽/год и не окупается при ~1 уведомлении
> в неделю. Если тебе нужна автозагрузка — можно дописать клиент.

## Что это НЕ делает

- Не подписывает XML электронной подписью (подпишешь в Диадоке при загрузке).
- Не ходит в ФНС напрямую.
- Не знает про МЧД автоматически — если подписываешь по доверенности,
  её реквизиты вписываешь в настройках.

## Установка

Нужен **Python 3.10+**.

```bash
git clone <url этого репозитория> wb-upd
cd wb-upd

python3 -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка

### 1. Токен WB API

Кабинет продавца WB → **Настройки → Доступ к API** → создай токен с правом
**«Документы»**. Скопируй и вставь в файл `.env`:

```bash
cp .env.example .env
# открой .env и впиши WB_API_TOKEN=<твой_токен>
```

### 2. Реквизиты

Запусти интерфейс и заполни вкладку **«Настройки»**:

```bash
streamlit run app.py
```

Откроется http://localhost:8501. На вкладке «Настройки» нужны:

- реквизиты ИП/ООО (ИНН, ОГРНИП, адрес)
- банк (БИК, р/с, корсчёт)
- налоговый режим и ставка НДС
- подписант (ФИО, способ подтверждения полномочий: без доверенности или МЧД)
- данные покупателя — по умолчанию **ООО РВБ** (Wildberries), трогать не надо

Всё это сохраняется локально в `settings.yaml`. Файл **не коммитится** в git.

## Использование

1. Вкладка **«Уведомления о выкупе»**:
   - выбираешь дату «С», по умолчанию — 1 января текущего года;
   - жмёшь **«⚡ Сформировать УПД для всех новых»** — прогоняет все
     необработанные уведомления;
   - **«📦 Скачать все»** — забирает все сгенерированные XML одним zip-архивом.

2. В Диадоке: «Создать документ» → «Загрузить из файла» → перетаскиваешь XML →
   подписываешь КЭП → отправляешь контрагенту.

3. Вкладка **«Журнал»** — история всех обработанных уведомлений.

## Где что лежит

| Файл / папка | Назначение |
|--------------|------------|
| `app.py` | Streamlit UI |
| `wb_client.py` | Клиент WB API |
| `xlsx_parser.py` | Парсер xlsx из архива уведомления |
| `upd_builder.py` | Генератор XML УПД (ФНС 5.03, СЧФДОП) |
| `config.py` | Схема настроек + чтение/запись `settings.yaml` |
| `storage.py` | SQLite-журнал обработанных документов |
| `.env` | Токен WB (**локально, не в git**) |
| `settings.yaml` | Реквизиты (**локально, не в git**) |
| `storage.db` | Журнал (**локально, не в git**) |
| `output/` | Сгенерированные XML УПД |

## Безопасность

- Токен WB и реквизиты хранятся **только у тебя на машине**, никуда не уходят.
- Инструмент ходит только к `common-api.wildberries.ru` и
  `documents-api.wildberries.ru`.
- Код открытый — можешь проверить сам.
